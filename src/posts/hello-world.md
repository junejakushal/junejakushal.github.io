This is my first post. I've been meaning to start writing for a while, and here we are.

## Why write?

Writing forces clarity. When I can't explain something simply, it usually means I don't understand it well enough yet. This blog is as much for me as it is for anyone who stumbles across it.

I plan to write about:

- Software I'm building and lessons learned
- Things I'm reading and thinking about
- Experiments, tools, and small discoveries

## How this works

Posts are written in plain Markdown files in the `src/posts/` directory. Adding a new `.md` file with frontmatter is all it takes to publish a new post — no CMS, no database, just files.

```bash
# Create a new post
touch src/posts/my-new-post.md
```

That's the whole workflow.
